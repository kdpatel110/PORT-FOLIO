# KD Portfolio

Personal portfolio built with Next.js and TypeScript.

## Overview

This repository contains a developer portfolio site showcasing projects, skills, experience, and contact information. It uses the Next.js `app/` router and a component-driven structure located in the `components/` folder.

## Tech

- Next.js
- TypeScript
- React
- CSS (global styles in `app/globals.css`)

## Project Structure

- `app/` — Next.js app routes and global layout
- `components/` — UI and page components
- `lib/` — data and utility helpers
- `public/images/` — static images

## Local Setup

1. Install dependencies:

```bash
pnpm install
```

2. Run development server:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build && pnpm start
```
